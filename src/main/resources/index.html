Руководство по использованию

Перед работой с бд ее необходимо создать. В модуле db в пакете org.java_courses запустите класс Main. В этом же пакете в классе CREDS хранятся креды для подключения к бд

Чтобы просмотреть товары нужно послать GET запрос такого вида http://localhost:3466/products/all

Для просмотра товаров определенной организации нужно послать GET запрос http://localhost:3466/products/getByOrganization/org2

POST запросы:

Добавление товара

В консоли запустить curl -H "Content-Type: application/json" -X POST -d "{\"product_name\":\"yellow\",\"organization_name\":\"renault\",\"amount\":123}" http://localhost:3466/products/add

Удаление товара

В консоли запустить curl -H "Content-Type: application/json" -X POST -d "yellow" http://localhost:3466/products/delete
